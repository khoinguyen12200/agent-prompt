# Workflow Skills (Highly Recommended)

## gstack

**Status:** HIGHLY RECOMMENDED  
**Source:** https://github.com/garrytan/gstack

**Install:**
```bash
git clone --single-branch --depth 1 https://github.com/garrytan/gstack.git .claude/skills/gstack
cd .claude/skills/gstack && ./setup
```

**What:** 23 specialist skills for end-to-end development — planning, review, QA, shipping

**Commands:**
- `/office-hours` — 6 forcing questions for product planning
- `/plan-ceo-review` — Strategic challenge
- `/plan-eng-review` — Architecture planning
- `/review` — Code review
- `/qa` — Browser testing
- `/ship` — Release workflow
- `/cso` — Security audit
- `/browse` — Real browser automation
- Plus 15 more...
