From: https://github.com/anthropics/skills + https://github.com/ComposioHQ/awesome-claude-skills

| Name | Source | Install | Description |
|------|--------|---------|-------------|
| docx | anthropics | `/plugin install document-skills@anthropic-agent-skills` | Create, edit, analyze Word docs with tracked changes, comments, formatting |
| pdf | anthropics | `/plugin install document-skills@anthropic-agent-skills` | Extract text, tables, metadata, merge & annotate PDFs |
| pptx | anthropics | `/plugin install document-skills@anthropic-agent-skills` | Read, generate, and adjust slides, layouts, templates |
| xlsx | anthropics | `/plugin install document-skills@anthropic-agent-skills` | Spreadsheet manipulation: formulas, charts, data transformations |
| markdown-to-epub | awesome | Clone repo | Converts markdown documents into professional EPUB ebook files. By @smerchek |
| csv-data-summarizer | awesome | Clone repo | Automatically analyzes CSV files and generates insights with visualizations. By @coffeefuelbump |
